Git_lg

git log 可以用来查看提交历史，但格式并不舒服，我们可以通过配置 git config 来解决这个问题：

运行

git config --global alias.lg 'log --pretty=format:"%h - %an, %ad : %s" --date=format:"%Y-%m-%d %H:%M:%S"'

之后，便可以使用 git lg 来查看精简版本的，更舒服的提交历史了，如本项目现在看的话会变成
